# StoneStoryQoL

## Features

    - Faster Fusing/Breaking/Chest opening animations.
    - Uncaps gamespeed(you'll need an external program/mod to change it for now, I'll add a feature to change it in-game later).
    - More StoneScript functions/variables.
        - `loc.restart` - Restarts the current level.
        - `foe.wakeupTics` - Gets the amount of tics until the foe wakes up.
        - `foe.isAwake` - Gets whether the foe is awake or not.
        - `player.level` - Gets the current level of the player. (Just a direct way to get the lvl without needing math)
        - `treasure.openAll` - Opens all treasures in your inventory. (VERY experimental, use with caution.)
